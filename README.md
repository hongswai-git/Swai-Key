# Swai Key

[中文](#中文)

A small, portable, ad-free Windows keyboard automation tool built with .NET Framework 4.8 and WPF.

![Swai Key screenshot](docs/screenshot-en.png)

## Features

- Supports selecting the target key
- Supports Ctrl / Shift / Alt key combinations
- Supports custom key press intervals
- Supports the global F7 hotkey to start / stop
- Supports always-on-top window mode
- Native WPF interface, no web wrapper
