### 这是什么

Swai Key 是一个 Windows 键盘自动按键工具。

我做它的原因很简单：有些重复按键操作不值得一直手动做，但我又不想要复杂、臃肿、广告很多的工具。Swai Key 只保留自动按键需要的核心功能，打开就能用。

### 主要功能

- 支持选择要自动按下的按键。
- 支持 Ctrl / Shift / Alt 组合键。
- 支持自定义按键间隔。
- 支持 F7 全局热键启动 / 停止。
- 支持窗口置顶。
- 绿色小工具，下载后直接运行。

### 怎么使用

1. 打开 `Swai Key.exe`。
2. 选择需要自动按下的按键。
3. 如果需要组合键，勾选 Ctrl / Shift / Alt。

### 适合用来做什么

- 重复触发某个键盘按键。

### 常见问题

**为什么按 F7 没反应？**

请确认 Swai Key 正在运行。如果其他软件占用了 F7，可能会导致快捷键冲突。

**可以在游戏里使用吗？**

技术上它只是发送键盘按键，但不同游戏和平台规则不同。请先确认你使用的软件或游戏是否允许这类工具。

**需要安装吗？**

不需要。Swai Key 是一个独立可执行文件，下载后直接运行即可。

I built it because some repeated key presses should not require a heavy or noisy tool. Swai Key keeps only the core controls needed for simple keyboard automation.

- Use Ctrl / Shift / Alt key combinations.
- Set a custom key press interval.
- Start or stop with the global F7 hotkey.

### FAQ

**Why does F7 not work?**

Make sure Swai Key is running. If another app is already using F7, the hotkey may conflict.

**Can I use it in games?**

Technically, Swai Key only sends keyboard input. Different games and platforms have different rules, so check whether this kind of tool is allowed before using it.

**Does it need installation?**

No. Swai Key is a standalone executable. Download it and run it directly.
