# 键盘连点器

一个小体积、免安装、无广告的 Windows 键盘自动按键工具，基于 .NET Framework 4.8 + WPF 开发。

![键盘连点器截图](docs/screenshot.png)

## 功能

- 支持选择目标按键
- 支持 Ctrl / Shift / Alt 组合按键
- 支持自定义按键间隔
- 支持 F7 全局热键启动 / 停止
- 支持窗口置顶
- 科技风 WPF 界面，不使用网页套壳
